1、git 客户端设置用户名
   >git config --global user.name 'chixinfushui'  
2、git 设置用户名邮箱
   >git config --global user.email '364105294@qq.com'
   
设置在github仓库主页显示谁提交了该文件

查看设置
>git config --list

在文件内初始化仓库：
>git init

git提交本地仓库
>git add filename
>git commit -m 'something'

git 克隆
>git clone 仓库地址

本地仓库同步到远程仓库
>git push
